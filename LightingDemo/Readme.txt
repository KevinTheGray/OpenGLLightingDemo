-----------------------------------------
Building:
-----------------------------------------
  - Open the folder kevin.joseph_CS300_1
  - Open the folder CS300_Assignment
  - Open the solution named CS300_Assignment.sln
  - Set mode to RELEASE MODE
  - Build the solution, it should compile.  It won't run fromm visual studios.
  - Close Visual Studios
  
-----------------------------------------
Running:
-----------------------------------------
  - Navigate back to the solution directory
  - Run CS300_Assignment.exe
  
----------------------------------------
Usage:
----------------------------------------
- Load files with the File Loader box.
- Play around with values in GLUI window.
- Play around with light values.
- To load a new file, you have to press ESC in the main window after entering the file name.
- To change the scale, you will have to press 's' in the main window after entering the new scale value.
- You can use the mouse to move the camera around.
